## 🚀 快速开始

1. 进入感兴趣的知识库目录，优先查看其 `README.md` 获取阅读顺序、难度分层与前置知识
2. 每篇文档均包含：核心原理、代码示例、面试题、实战案例
3. 文档间通过交叉引用（`> 📖` 标记）互相链接，支持知识间跳转

## 📝 贡献规范

### 新增知识库

1. 在根目录创建新的领域文件夹（如 `math/`、`finance/`）
2. 在文件夹内创建 `README.md` 作为该领域的导航索引
3. 更新本文件的知识库索引表

### 文档编写规范

- 文件命名：`编号-主题名.md`（如 `01-Go语言深入.md`）
- 文档结构：标题 → 目录 → 核心章节 → 面试题 → 实战案例 → 总结
- 交叉引用：使用 `> 📖` 引用块 + 相对路径链接
